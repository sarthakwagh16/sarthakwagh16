## 🚀 Projects

### 1. 🏦 IFRS9 PD Scorecard — Personal Loan Portfolio
> *End-to-end credit risk model from raw bureau data to regulatory deployment*

**Problem:** Lender needed a robust, IFRS9-compliant probability of default model to replace an underperforming champion scorecard for personal loan origination.

**Approach:**
- Extracted and processed bureau + behavioural features from Hadoop; applied WOE/IV binning and feature selection
- Trained ensemble models (Logistic Regression, Decision Trees, Random Forest) with hyperparameter tuning
- Validated using KS statistic, Gini coefficient, and out-of-time (OOT) holdout samples
- Documented in line with SR 11-7 model risk management standards

**Outcome:** Challenger model achieved KS 59.1 and Gini 65.08% — clearing the production threshold — with projected **8–12% ECL reduction** post-deployment

`Python` `Scikit-learn` `WOE/IV` `Logistic Regression` `Random Forest` `IFRS9` `SR 11-7` `Hadoop`

---

### 2. 🚗 ECL Framework — Auto Loan Book (PD · LGD · EAD)
> *Full Expected Credit Loss computation pipeline for regulatory provisioning*

**Problem:** Auto loan portfolio lacked segment-level loss estimates, leading to inaccurate provisioning and elevated model risk.

**Approach:**
- Built three interconnected models: PD (classification), LGD (regression on recovery data), EAD (exposure at default via CCF estimation)
- Applied vintage analysis and cohort segmentation to identify structural risk segments
- Performed back-testing and benchmarking against industry LGD/EAD norms

**Outcome:** Improved ECL estimation accuracy and enabled **granular, audit-ready provisioning** aligned with IFRS9 Stage 1/2/3 logic

`Python` `SQL` `IFRS9` `Vintage Analysis` `Cohort Segmentation` `Model Governance`
